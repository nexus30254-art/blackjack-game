🃏 Blackjack Web Game
A sleek, responsive Blackjack game built entirely with HTML5, CSS3, and Vanilla JavaScript. This project focuses on managing game state and updating the UI dynamically without a backend.

🛠️ Built With
HTML5: Structured the game board, player areas, and controls.

CSS3: Styled the layout with a focus on a "casino" feel, including responsive design for mobile and desktop.

Vanilla JavaScript: * DOM Manipulation: Real-time updates of scores and card displays.

Game Logic: Handling the deck shuffle, "Hit" vs "Stand" logic, and win/loss calculations.

State Management: Tracking the player's hand and the dealer's hidden card.

🎮 Features
Dynamic UI: The screen updates instantly when cards are drawn—no page refreshes required.

Responsive Design: Playable on mobile, tablet, or desktop browsers.

Win/Loss Animations: Visual feedback for "Bust," "Win," or "Draw" scenarios.

Automated Dealer: A JavaScript-powered dealer that follows house rules (e.g., stands on 17).
